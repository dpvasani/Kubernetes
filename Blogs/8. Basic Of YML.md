### ğŸ“š 13. RBAC (Role, RoleBinding)

```yaml

  namespace: default
  name: pod-reader

  - apiGroups: [""]
    resources: ["pods"]

```

```yaml
# RoleBinding

metadata:
  name: read-pods-binding
  namespace: default
subjects:
  - kind: ServiceAccount
    name: my-serviceaccount
    namespace: default
roleRef:
  kind: Role
  name: pod-reader
  apiGroup: rbac.authorization.k8s.io
```
